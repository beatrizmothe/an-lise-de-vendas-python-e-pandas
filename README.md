# Análise de Vendas com Python

Esse projeto é um estudo de análise de vendas usando Python e Pandas.  

---

## O que o projeto faz

- Calcula a **receita** de cada venda 
- Filtra vendas por **categoria**  
- Mostra o **produto mais vendido**  
- Mostra a **região com maior faturamento**  
- Gera um arquivo **Excel**
